# Surface Composer

The Surface Composer extends the Flex Composer to provide a specialized API for creating surface-like components with consistent padding, border radius, and dimensions. It's perfect for creating cards, buttons, and other container elements.

## Basic Usage

```tsx
import styled from "styled-components";
import { surface } from "stylings";

const Card = styled.div`
  ${surface({ paddingX: 2, paddingY: 1, radius: 1 })};
`;
```

## Configuration

Define surface properties through configuration:

```tsx
// Basic configuration
${surface({
  paddingX: 2,
  paddingY: 1,
  radius: 1,
  height: 3,
  width: 20
})}
```

## Padding

Control padding on surfaces:

```tsx
// All sides
${surface({ paddingX: 2, paddingY: 1 }).padding}

// Horizontal padding
${surface({ paddingX: 2 }).paddingX}

// Vertical padding
${surface({ paddingY: 1 }).paddingY}

// Remove padding on specific sides
${surface({ paddingX: 2 }).noPT} // No padding top
${surface({ paddingX: 2 }).noPB} // No padding bottom
${surface({ paddingX: 2 }).noPL} // No padding left
${surface({ paddingX: 2 }).noPR} // No padding right
```

## Border Radius

Control border radius:

```tsx
// Custom radius
${surface({ radius: 1 }).radius}

// Circular shape
${surface({ radius: 1 }).circle}
```

## Dimensions

Control surface dimensions:

```tsx
// Height
${surface({ height: 3 }).height}

// Width
${surface({ width: 20 }).width}

// Both height and width
${surface({ height: 3, width: 20 }).size}
```

## Flex Properties

Since Surface Composer extends Flex Composer, you can use all flex properties:

```tsx
// Combine surface properties with flex properties
${surface({ paddingX: 2 }).row.alignCenter}
${surface({ radius: 1 }).column.gap(2)}
```

## Example Components

Here's a practical example of how to use the Surface Composer in real components:

```tsx
const Card = styled.div`
  ${surface({ paddingX: 2, paddingY: 1, radius: 1 })};
`;

const Button = styled.button`
  ${surface({ paddingX: 2, paddingY: 1, radius: 0.5, height: 2.5 })};
`;

const Avatar = styled.div`
  ${surface({ size: 3, radius: 1 }).circle};
`;

const Input = styled.input`
  ${surface({ paddingX: 2, paddingY: 1, radius: 0.5, height: 2.5 })};
`;

const Modal = styled.div`
  ${surface({ paddingX: 3, paddingY: 2, radius: 2, width: 30 })};
`;
```

The Surface Composer makes it easy to create consistent surface-like components across your application. It combines the power of flexbox layout with specialized surface properties, making it perfect for creating UI components that need both layout and surface styling.
