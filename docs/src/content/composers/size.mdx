# Size Composer

The Size Composer provides a comprehensive API for managing dimensions, spacing, and transformations in your components. It offers a fluent interface for controlling various size-related properties with support for different units and scaling systems.

## Basic Usage

```tsx
import styled from "styled-components";
import { size } from "styledwind";

const Box = styled.div`
  ${size.width(2).height(2)};
`;
```

## Units

Specify sizes using different units:

```tsx
// Base unit (scaled by 4)
${size.base(1)} // 0.25rem

// Rem units
${size.rem(2)} // 2rem

// Level-based scaling
${size.level(2)} // Scaled by level system

// Pixel units
${size.px(16)} // 16px

// Em units
${size.em(1.5)} // 1.5em
```

## Dimensions

Control element dimensions:

```tsx
// Width
${size.width(2)}

// Height
${size.height(2)}

// Both width and height
${size.size(2)}

// Minimum dimensions
${size.minWidth(2)}
${size.minHeight(2)}
${size.minSize(2)}

// Maximum dimensions
${size.maxWidth(2)}
${size.maxHeight(2)}
${size.maxSize(2)}
```

## Margins

Add margins to elements:

```tsx
// All sides
${size.margin(2)}

// Horizontal margins
${size.marginX(2)}

// Vertical margins
${size.marginY(2)}

// Individual sides
${size.marginTop(2)}
${size.marginBottom(2)}
${size.marginLeft(2)}
${size.marginRight(2)}
```

## Padding

Add padding to elements:

```tsx
// All sides
${size.padding(2)}

// Horizontal padding
${size.paddingX(2)}

// Vertical padding
${size.paddingY(2)}

// Individual sides
${size.paddingTop(2)}
${size.paddingBottom(2)}
${size.paddingLeft(2)}
${size.paddingRight(2)}
```

## Gap

Add gap between elements:

```tsx
// Gap between grid or flex items
${size.gap(2)}
```

## Transformations

Apply translations:

```tsx
// Translate X
${size.transformX(2)}

// Translate Y
${size.transformY(2)}

// Translate both X and Y
${size.transformXY(2)}
```

## Example Components

Here's a practical example of how to use the Size Composer in real components:

```tsx
const Card = styled.div`
  ${size.width(20).padding(2)};
`;

const Button = styled.button`
  ${size.paddingX(2).paddingY(1)};
`;

const Grid = styled.div`
  ${size.gap(2)};
`;

const Modal = styled.div`
  ${size.width(30).maxHeight(40).padding(3)};
`;

const Icon = styled.div`
  ${size.size(1.5).transformY(-0.25)};
`;
```

The Size Composer makes it easy to manage dimensions and spacing across your application while maintaining consistency through its unit system. It provides a comprehensive set of tools for controlling all aspects of element sizing and positioning.
