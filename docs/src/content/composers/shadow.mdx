# Shadow Composer

The Shadow Composer provides a fluent API for creating box shadows in your components. It allows you to control all aspects of shadow styling, including offset, blur, spread, color, and opacity.

## Basic Usage

```tsx
import styled from "styled-components";
import { shadow } from "styledwind";

const Card = styled.div`
  ${shadow.x(0).y(2).blur(4).color("rgba(0, 0, 0, 0.1)")};
`;
```

## Shadow Properties

Control various shadow properties:

```tsx
// Horizontal offset
${shadow.x(2)}

// Vertical offset
${shadow.y(2)}

// Blur radius
${shadow.blur(4)}

// Shadow color
${shadow.color("rgba(0, 0, 0, 0.1)")}

// Shadow spread
${shadow.spread(1)}

// Inset shadow
${shadow.inset(true)}

// Shadow opacity (requires color to be set first)
${shadow.color("#000000").opacity(0.1)}
```

## Common Shadow Patterns

Here are some common shadow patterns you might want to use:

```tsx
// Subtle elevation
${shadow.x(0).y(1).blur(2).color("rgba(0, 0, 0, 0.1)")}

// Medium elevation
${shadow.x(0).y(2).blur(4).color("rgba(0, 0, 0, 0.1)")}

// High elevation
${shadow.x(0).y(4).blur(8).color("rgba(0, 0, 0, 0.1)")}

// Inset shadow (for pressed state)
${shadow.x(0).y(1).blur(2).color("rgba(0, 0, 0, 0.1)").inset(true)}

// Colored shadow
${shadow.x(0).y(2).blur(4).color("rgba(0, 123, 255, 0.2)")}
```

## Example Components

Here's a practical example of how to use the Shadow Composer in real components:

```tsx
const Card = styled.div`
  ${shadow.x(0).y(2).blur(4).color("rgba(0, 0, 0, 0.1)")};
`;

const Button = styled.button`
  ${shadow.x(0).y(1).blur(2).color("rgba(0, 0, 0, 0.1)")};
  
  &:active {
    ${shadow.x(0).y(1).blur(2).color("rgba(0, 0, 0, 0.1)").inset(true)};
  }
`;

const Modal = styled.div`
  ${shadow.x(0).y(4).blur(8).color("rgba(0, 0, 0, 0.2)")};
`;

const FloatingButton = styled.button`
  ${shadow.x(0).y(2).blur(4).color("rgba(0, 123, 255, 0.2)")};
`;
```

The Shadow Composer makes it easy to create consistent elevation and depth across your application. It provides a clean API for controlling all aspects of box shadows, making it perfect for creating modern, layered interfaces.
