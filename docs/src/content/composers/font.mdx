# Font Composer

The Font Composer provides a comprehensive API for managing typography in your components. It's designed to encourage creating reusable, composable typography styles that maintain consistency across your application.

## Basic Usage

```tsx
import styled from "styled-components";
import { font } from "styledwind";

const Heading = styled.h1`
  ${font.family("Inter, sans-serif").size(2).weight(700)};
`;
```

## Creating Reusable Typography Styles

Start by defining base typography styles that you can compose into more specific styles:

```tsx
// Define base font styles
const baseFont = font
  .family("Inter, sans-serif")
  .lineHeight(1.5)
  .antialiased;

// Compose into more specific styles
export const typo = {
  // Headings
  h1: baseFont.size(2.5).weight(700).lineHeight(1.2),
  h2: baseFont.size(2).weight(600).lineHeight(1.3),
  h3: baseFont.size(1.5).weight(600).lineHeight(1.4),
  
  // Body text
  body: baseFont.size(1),
  small: baseFont.size(0.875),
  
  // Special cases
  label: baseFont.size(0.875).weight(500),
  caption: baseFont.size(0.75).secondary,
};
```

## Font Properties

Control basic font properties:

```tsx
// Font family
${font.family("Inter, sans-serif")}

// Font size
${font.size(1.5)} // 1.5em
${font.size("2rem")}

// Font weight
${font.weight(700)}
// Or use predefined weights
${font.w100} // Thin
${font.w200} // Extra Light
${font.w300} // Light
${font.w400} // Regular
${font.w500} // Medium
${font.w600} // Semi Bold
${font.w700} // Bold
${font.w800} // Extra Bold
${font.w900} // Black
${font.normal} // Alias for w400
```

## Line Height

Control line height with various options:

```tsx
// Custom line height
${font.lineHeight(1.5)}

// Predefined line heights
${font.copyLineHeight} // 1.5 (good for body text)
${font.headingLineHeight} // 1.25 (good for headings)
${font.resetLineHeight} // 1 (reset to normal)
```

## Text Transform

Transform text case:

```tsx
// Uppercase
${font.uppercase}

// Lowercase
${font.lowercase}

// Capitalize
${font.capitalize}
```

## Text Decoration

Add text decorations:

```tsx
// Underline
${font.underline}
```

## Text Alignment

Control text alignment:

```tsx
// Left align
${font.left}

// Center align
${font.center}

// Right align
${font.right}
```

## Text Overflow

Handle text overflow:

```tsx
// Ellipsis for single line
${font.ellipsis}

// Prevent wrapping
${font.nowrap}

// Limit to specific number of lines
${font.maxLines(3)}
```

## Letter Spacing

Control letter spacing:

```tsx
// Custom letter spacing
${font.letterSpacing(0.05)} // 0.05em
```

## Text Rendering

Improve text rendering:

```tsx
// Enable antialiasing
${font.antialiased}

// Balance text wrapping
${font.balance}
```

## Opacity Variants

Create different opacity variants:

```tsx
// Secondary opacity (50%)
${font.secondary}

// Tertiary opacity (30%)
${font.tertiary}
```

## Example Components

Here's a practical example of how to compose typography styles in real components:

```tsx
// Define base styles
const baseFont = font
  .family("Inter, sans-serif")
  .lineHeight(1.5)
  .antialiased;

// Compose into specific styles
const typo = {
  heading: baseFont.size(2).weight(700).headingLineHeight,
  body: baseFont.size(1).copyLineHeight,
  label: baseFont.size(0.875).weight(500),
};

// Use in components
const Card = styled.div`
  ${typo.body};
`;

const CardTitle = styled.h2`
  ${typo.heading};
`;

const FormLabel = styled.label`
  ${typo.label};
`;

const TruncatedText = styled.div`
  ${typo.body.ellipsis};
`;

const MultiLineText = styled.div`
  ${typo.body.maxLines(3)};
`;
```

The Font Composer makes it easy to create a consistent typography system across your application. By defining base styles and composing them into more specific styles, you can maintain consistency while keeping your code DRY and maintainable.
