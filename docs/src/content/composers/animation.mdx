# Animation Composer

The Animation Composer provides a powerful API for creating complex animations in your components. It supports keyframe animations, transforms, filters, and various animation properties with a fluent interface.

## Basic Usage

```tsx
import styled from "styled-components";
import { animation } from "stylings";

const FadeIn = styled.div`
  ${animation.fadeIn.duration(200)};
`;
```

## Predefined Animations

Use common animation patterns:

```tsx
// Fade in
${animation.fadeIn}

// Fade out
${animation.fadeOut}

// Slide up from bottom
${animation.slideUpFromBottom("20px")}

// Slide down from top
${animation.slideDownFromTop("20px")}

// Slide left from right
${animation.slideLeftFromRight("20px")}

// Slide right from left
${animation.slideRightFromLeft("20px")}

// Zoom in
${animation.zoomIn(1.2)}

// Zoom out
${animation.zoomOut(0.8)}

// Spin
${animation.spin}

// Infinite animation
${animation.infinite}
```

## Custom Properties

Create custom animations with specific properties:

```tsx
// Opacity animation
${animation.opacity([0, 1])}

// Transform animations
${animation.x([0, 20])}
${animation.y([0, -20])}
${animation.scale([1, 1.2])}
${animation.rotate(["0deg", "45deg"])}

// Filter animations
${animation.blur(["0px", "4px"])}
```

## Animation Configuration

Control animation behavior:

```tsx
// Duration
${animation.duration(200)}

// Delay
${animation.delay(100)}

// Easing function
${animation.easing("ease-in-out")}

// Fill mode
${animation.fillMode("forwards")}

// Iteration count
${animation.iterationCount(2)}
```

## Transform Style

Control how transforms are rendered:

```tsx
// Use 3D transforms
${animation.transformStyle("preserve-3d")}
```

## Complex Animations

Create complex animations by combining multiple properties:

```tsx
// Fade in and slide up
${animation
  .opacity([0, 1])
  .y([20, 0])
  .duration(300)
  .easing("ease-out")}

// Scale and rotate
${animation
  .scale([1, 1.2])
  .rotate(["0deg", "45deg"])
  .duration(200)
  .easing("ease-in-out")}

// Multiple transforms
${animation
  .x([0, 20])
  .y([0, -20])
  .scale([1, 1.1])
  .duration(300)}
```

## Example Components

Here's a practical example of how to use the Animation Composer in real components:

```tsx
const Modal = styled.div`
  ${animation
    .opacity([0, 1])
    .scale([0.95, 1])
    .duration(200)
    .easing("ease-out")};
`;

const Button = styled.button`
  ${animation
    .scale([1, 1.05])
    .duration(150)
    .easing("ease-in-out")};
    
  &:active {
    ${animation
      .scale([1.05, 0.95])
      .duration(100)
      .easing("ease-in-out")};
  }
`;

const LoadingSpinner = styled.div`
  ${animation.spin.duration(1000).infinite};
`;

const Notification = styled.div`
  ${animation
    .slideRightFromLeft("100%")
    .duration(300)
    .easing("ease-out")};
`;
```

The Animation Composer makes it easy to create sophisticated animations across your application. It provides a comprehensive set of tools for controlling all aspects of animations, from simple transitions to complex keyframe-based animations.
