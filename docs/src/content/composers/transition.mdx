# Transition Composer

The Transition Composer provides a fluent API for creating smooth transitions in your components. It allows you to control transition properties, timing, and easing functions, making it easy to create polished animations.

## Basic Usage

```tsx
import styled from "styled-components";
import { transition } from "styledwind";

const Button = styled.button`
  ${transition.duration(150).easing("ease-in-out")};
`;
```

## Transition Properties

Control which properties should transition:

```tsx
// All properties
${transition.all}

// Color-related properties
${transition.colors}

// Common properties (colors, opacity, shadows, transforms, filters)
${transition.common}

// Custom properties
${transition.property("width", "height")}
```

## Timing

Control transition timing:

```tsx
// Duration
${transition.duration(150)} // 150ms
${transition.duration("200ms")}

// Slow release (longer duration on hover)
${transition.duration(150).slowRelease}
```

## Easing Functions

Control transition easing:

```tsx
// Linear
${transition.easing("linear")}

// Ease-in-out
${transition.easing("ease-in-out")}

// Custom cubic-bezier
${transition.easing("cubic-bezier(0.4, 0, 0.2, 1)")}
```

## Common Transition Patterns

Here are some common transition patterns you might want to use:

```tsx
// Smooth color transitions
${transition.colors.duration(150).easing("ease-in-out")}

// Smooth transform transitions
${transition.property("transform").duration(200).easing("cubic-bezier(0.4, 0, 0.2, 1)")}

// Smooth opacity transitions
${transition.property("opacity").duration(150)}

// Smooth all transitions with slow release
${transition.all.duration(200).slowRelease}
```

## Example Components

Here's a practical example of how to use the Transition Composer in real components:

```tsx
const Button = styled.button`
  ${transition.colors.duration(150).easing("ease-in-out")};
  
  &:hover {
    background-color: #0056b3;
  }
`;

const Card = styled.div`
  ${transition.common.duration(200)};
  
  &:hover {
    transform: translateY(-2px);
    box-shadow: 0 4px 8px rgba(0, 0, 0, 0.1);
  }
`;

const Modal = styled.div`
  ${transition.property("opacity", "transform").duration(200)};
  
  &.entering {
    opacity: 0;
    transform: scale(0.95);
  }
  
  &.entered {
    opacity: 1;
    transform: scale(1);
  }
`;

const Toggle = styled.div`
  ${transition.property("width", "height").duration(150).easing("ease-in-out")};
`;
```

The Transition Composer makes it easy to create smooth, professional animations across your application. It provides a clean API for controlling all aspects of transitions, making it perfect for creating polished, interactive interfaces.
